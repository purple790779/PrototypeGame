# 성주 경영 시뮬레이션 (MVP)

이 프로젝트는 “성주(영주)+성/마을 경영 시뮬레이션” 기획서의 **MVP**를 정적 웹(HTML/CSS/JS)로 구현한 스캐폴딩입니다.

## 실행
- 폴더 안의 `index.html`을 브라우저로 열면 실행됩니다.
- 더 편하게 하려면 로컬 서버(예: VSCode Live Server, `python -m http.server`)로 열어도 됩니다.

## 포함된 시스템 (MVP)
- 일 단위 진행(하루 진행 / 자동 진행)
- 핵심 지표: 인구/수용, 노동력, 식량, 목재, 돌, 금고, 행복, 치안, 정통성
- 노동 배치: 농부/목수/건설/경비
- 건설: 주거/농장/저장고/성벽/선술집/의무소
- 세금: 주간 징수 + 일 단위 압박(행복/정통성)
- 이벤트: 약탈/가뭄/상단/전염병 (선택지형 모달)
- 저장/불러오기: localStorage

## 다음 확장 추천
- 물류(운송/도로)와 병목
- 파벌(민중/길드/교회/봉신) 및 정책 트리
- 인물(집사/재상/주교 등) + 관계 기반 이벤트
